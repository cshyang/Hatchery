# PR Summary

Write the PR body for a human reviewer.

## Include

- What changed.
- Why it changed.
- Tests run.
- Known risks or follow-ups.

## Exclude

- Private chain-of-thought.
- Secrets, tokens, or environment values.
- Internal runner logs unless needed to explain a failure.
