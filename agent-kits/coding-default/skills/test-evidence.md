# Test Evidence

Record what was tested and what it proves.

## Checklist

- Run the smallest focused tests first.
- Run the repo's standard typecheck and test commands when the change touches shared behavior.
- Capture failures with the exact command and shortest useful error.
- Do not claim a command passed unless it was run in the runner environment.

## Output

- Commands run.
- Pass/fail status.
- Evidence summary.
- Remaining gaps.
